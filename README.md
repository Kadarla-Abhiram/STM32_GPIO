# STM32 GPIO Projects

## Overview

This repository contains basic STM32 GPIO projects developed using **STM32CubeIDE** and the **HAL library**. The projects demonstrate digital input/output programming and fundamental GPIO concepts for embedded systems.

## Hardware

- STM32F103C8T6 (Blue Pill)
- LEDs
- Push Button
- Breadboard
- Jumper Wires

## Software

- STM32CubeIDE
- STM32 HAL Library

## Projects

- ✅ LED Blink *(Completed)*
- ⏳ Push Button Interface
- ⏳ LED Toggle
- ⏳ Traffic Light Controller
- ⏳ Multiple LED Patterns
- ⏳ Button Debouncing

## Concepts Covered

- GPIO Configuration
- Digital Input/Output
- HAL GPIO Functions
- Embedded C Programming
